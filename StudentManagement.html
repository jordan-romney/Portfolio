This Student Management program was a project from my Software Development Degree. This project highlights various C++ concepts including:

  • Variables and Data Types
  • Control Structures
  • Arrays
  • Pointers and Memory
  • Functions
  • Object-Oriented Programming

This was my first major programming project for my degree program. All my classes leading up to this were mostly theory about object-oriented programming, data structures, and algorithms. Finally putting all that knowledge to use was very refreshing and incredibly frustrating. This project allowed me to learn how to shift my brain to think more like a programmer.

The opportunity to use classes and objects helped me to realize how beneficial object-oriented programming is. I was able to organize my code in such a way that I was able to keep the functionality of the program straight and feel confident while debugging and troubleshooting.

The most confusing part of this project was not being able to use vectors per the assignment requirements. Using basic arrays felt very clunky but it truly helped me to understand pointers and how C++ interacts with memory. Overall, this project is what helped me to understand I could truly be a developer.
